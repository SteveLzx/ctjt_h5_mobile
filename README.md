### ESLint from airbnb

1. js/ts 语句后使用英文分号结尾(;)
2. 只读常量使用 const 声明,局部变量使用 let 声明
3. 注释//后面空格
4. 不要直接修改参数值,需要修改时用变量赋值后修改
5. 声明类型(type)和接口(interface)时首字母大写
6. 使用箭头函数,直接 retrun 结果的无需写 return
